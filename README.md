# ⚡ Zeus WhatsApp Bot 

Zeus es un Bot de WhatsApp construido con Baileys v6.7.9 diseñado para otorgar la mejor experiencia al usuario permitiendo desde las descargas de contenido multimedia hasta la moderación de grupos.

## ✨ Características

- 🚀 Procesa y responde mensajes con gran velocidad 
- 💰 Sistema de economía con juegos y actividades para ganar monedas 
- 🔄 Actualizaciones constantes para mejorar y añadir nuevos comandos 
- 🔒 Administra tus grupos con facilidad 
- 🖼️ Sistema de bienvenida y despida automática para tus grupos 
- 🎮 Variedad de juegos como: tictactoe, ahorcados, blackjack y akinator

## 🎮 Comandos 

Zeus ofrece una variedad de comandod para la mejor experiencia del usuario:

- `/play` - Busca videos en Youtube 
- `/ytmp4` - Descarga videos de YouTube 
- `/ytmp3` - Descarga audio de videos de YouTube 
- `/igdl` - Descarga publicaciones de Instagram 
- `/tiktokdl` - Descarga publicaciones de TikTok
- `/twitter` - Descarga publicaciones de Twitter(X)
- `/Threads` - Descarga publicaciones de Threads
- `/menu` - Envía la lista de comandos disponibles 

...y muchos más!

## 🚀 Como Empezar

Para empezar a usar a Zeus:

1. [Click here para abrir un chat con zeus](https://wa.me/595985529728?text=/menu)
2. Usa `/menu` para ver todos los comandos disponibles 

## 🛠️ Tecnologías utilizadas

- [Baileys](https://github.com/whiskeysockets/baileys) v6.7.9 - Una potente biblioteca de JavaScript para interactuar con WhatsApp 
- [Node.js](https://nodejs.org/) - Es un entorno de ejecución de JavaScript basado en el motor V8 de Google Chrome
